Passo a passo para enviar um commit para o repositorio com comandos

1° Denominar o novo commit usando // git commit -m ''novo nome''  

2° Inserir o novo commit no github// git push


Branch

git checkout -b <branch_name> - criar uma nova branch
git checkout <branch_name> - vai para uma branch especifica

4° vai abrir uma mensagem que pode ser fechada
5° git push origin
